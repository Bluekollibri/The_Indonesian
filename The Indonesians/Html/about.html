About Project

About Project

The indonesian web adalah sebuah proyek yang mengandung informasi tentang indonesia jadi indonesian web ini disebarluaskan kepada seluruh indonesia, maka dari itu web ini masih versi satu supaya ketika web ini di improve dengan kreatif supaya design makin menarik, oh ya web ini perlu dijadikan project terakhir teknologi multi media, maka artikel untuk tutorial ada di link tersebut

Pembuatan Web

Untuk pembuatan web tersebut maka perlunya pengunaan bootstrap versi 5.3 tau 5.2.3, perlu juga pemahaman css, dan html, maka pembuatan pun sudah terbiasa dengan web-web yang sudah kalian menguasai tersebut, maka web ini jadi kreasi menurut kalian tersebut.

Profile Project

Pembuatan web ini dibuat oleh Bryan Matthews Justandi dari Universitas Kristen Maranatha yang berjurusan Teknik Informatika, dan bekelahiran 26 Januari 2003, pembuatan ini sesuai dengan kreasi saya sendiri, maka kreasi ditunjukan dengan kasus-kasu yang berbeda.
